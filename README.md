# Absher Pro Delegation – Smart Government Delegation Assistant (MVP)

## Short Description
Absher Pro Delegation هو نموذج أولي (MVP) لتحسين تجربة التفويض للخدمات الحكومية السعودية عبر منصات أبشر، أبشر أعمال، وناجز. يدعم دورين أساسيين: المانح (مانح التفويض) والمفوَّض له. يتضمن مساعدًا ذكياً، خيارات إمكانية وصول، ولوحة تحليلات مبسطة.

## Features
### Grantor Features
- إنشاء تفويض جديد عبر مسار إرشادي من 4 خطوات (اختيار الخدمة والمنصة، بيانات المفوَّض: الاسم/الهوية/الجوال مع تطبيع الأرقام للإنجليزية، تحديد المدة).
- عرض التفويضات حسب الحالة: قيد الموافقة، نشط، مكتمل، مرفوض، منتهي.
- أرشفة وحذف تفويضات (حذف منطقي لكل مستخدم).

### Delegate Features
- استعراض كل التفويضات الموجَّهة للمستخدم كمفوَّض.
- فتح كل تفويض لقراءة الشروط والأحكام بخط واحد واضح (RTL/LTR) مع قبول/رفض فوري وتسجيل الموافقة.
- قبول أو رفض التفويض مع تحديث فوري للحالة للطرفين.

### System & AI Features
- دعم منصات متعددة: أبشر، أبشر أعمال، ناجز.
- مساعد ذكاء اصطناعي محلي لإنشاء التفويضات باللغتين العربية والإنجليزية، مع اقتراح المنصة/الخدمة والمدة حسب النية.
- تبويب إمكانية الوصول (اللغة، تكبير النص، تباين عالٍ، أزرار أكبر) مضاف إليه تدفق طلب مندوب موثّق مع طلب موقع المستخدم وأجرة خدمة 200 ﷼.
- تبويب التحليلات مع بطاقات/رسوم (إجمالي التفويضات، توزيع الحالات، مدد الاستخدام، استخدام المنصات).
- تصميم RTL أولًا مع واجهة عربية وزر تبديل للإنجليزية.

## Tech Stack
- Frontend: React / Next.js (App Router)، TypeScript، Tailwind CSS مع دعم RTL.
- State management: React hooks أو مكتبة خفيفة (مثلاً Zustand/Context).
- Charts: مكتبة بسيطة للرسوم (sparkline أو دائرية/أعمدة).
- AI: طبقة وهمية قابلة للاستبدال بمزود LLM.
- Testing: يُنصح بـ Playwright أو Cypress لاختبارات E2E.

## Running Locally
- التثبيت: `npm install`
- التشغيل: `npm run dev`
- العنوان المحلي: http://localhost:3000
- متغيرات البيئة: أضف قسماً لـ `.env` عند الحاجة (مثل مفاتيح مزود AI أو API حقيقية لاحقاً).
- اختبارات E2E (Playwright): `npm run test:e2e` (قد تحتاج أول مرة لتشغيل `npx playwright install` لتثبيت المتصفحات).
  - تغطي إنشاء التفويض، قبول/رفض التفويضات، النصوص RTL/LTR، إعادة ضبط الحالة عند إعادة فتح النوافذ، تدفق اللغة الإنجليزية، تطبيع الأرقام، وتدفق طلب المندوب الموثّق (موقع + تكلفة 200 ﷼).
